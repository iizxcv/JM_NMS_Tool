## Backend, DB 구축

### DB 작업
```
# PostgreSQL 설치 및 접속
sudo apt update && sudo apt install -y postgresql
sudo -u postgres psql

# 두 개의 DB 생성 (주명님의 설계 반영)
CREATE USER admin WITH PASSWORD 'password123';
CREATE DATABASE web_db OWNER admin;  # 계정/권한/웹 로그용
CREATE DATABASE log_db OWNER admin;  # SNMP/Syslog/Config 정보용
\q

```
